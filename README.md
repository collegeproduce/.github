# College Produce

Internal development repository for **College Produce** resources.

This repo contains internal documentation and references used by engineering, product, and design teams to ensure consistency in product direction, messaging, and implementation decisions concerning CollegeProduce.

## Overview

**College Produce** is a litigation software company built by trial professionals with **25+ years of courtroom experience**.
- Our flagship product, ExhibitPro, automates exhibit tab and label creation by pulling titles directly from documents, turning hours of manual work into minutes. 
- Built by people who spent 25 years preparing trial binders ourselves. 
- We create tools we wish we'd had.

## Repository Purpose

This repository exists to support software development and engineering operations by providing:

- Internal technical documentation for ExhibitPro architecture, workflows, and core modules
- Engineering standards for code structure, naming conventions, and best practices
- DevOps and infrastructure references (deployment, environments, CI/CD, monitoring)
- Configuration guidelines for local development, staging, and production environments
- Shared implementation notes to ensure consistency across features and services
- Technical context behind key product decisions to support maintainability and scaling
